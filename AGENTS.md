# AGENTS.md

## Project Rules

- We use Paper and the Paper MCP to design the app experience.
- We use the components in `src/components/ui/` to implement and bring those designs to life in code.
- Do not edit files in `src/components/ui/`.
- Components in `src/components/ui/` are our shadcn components and should be treated as vendor-managed.
- If behavior or styling changes are needed, implement them outside `src/components/ui/` (for example: wrappers, composition, page-level styles, or new components in a different directory).
